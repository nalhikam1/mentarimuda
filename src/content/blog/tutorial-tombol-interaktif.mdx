Selamat datang di jurnal belajar **Mentari Muda**! Hari ini kita akan membedah salah satu fitur paling revolusioner di Svelte 5, yaitu **Runes**. Kita akan belajar membuat sebuah tombol yang bisa menghitung klik secara reaktif.

### 1. Perbandingan: Svelte vs Vanilla JS
Svelte memangkas kode yang membosankan. Cukup fokus pada data, layar akan mengikuti.

#### Cara Svelte 5 (Otomatis)
```javascript
let count = $state(0);
function handleClick() {
  count += 1;
}
```

#### Cara Biasa (Manual)
```javascript
let count = 0;
const element = document.getElementById('count');
function handleClick() {
  count += 1;
  element.innerText = count;
}
```

### 3. Simulasi Interaktif
Coba edit kode di bawah ini untuk melihat hasil kodingan di atas.

### 4. Dibalik Layar: Svelte & Astro
Astro bertugas mengambil komponen Svelte kamu, lalu mengubahnya menjadi file statis yang super kencang. `client:load` di atas memberitahu Astro: *"Jalankan JavaScript komponen ini saat halaman selesai dimuat"*.
